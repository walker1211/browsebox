# browsebox

browsebox is a Go CLI for launching isolated, proxy-routed Google Chrome sessions through a temporary mihomo controller. It can list available proxy nodes from your existing Clash Verge Rev/mihomo controller and run either temporary or persistent isolated browser sessions without mutating your main selector from `nodes`.
